Display and Logging Module
==========================
Overview
========
The display and logging module is a compute system for a street/track car that is intended to operate an LCD screen to provide live instrumentation, as well as a data logger.  Additional advanced functions will be integrated, such as GNSS lap timer functionality or other infotainment functions.

Hardware Description
====================
Raspberry Pi Compute
--------------------
- Raspberry Pi 5 (8MB RAM)
    - Raspberry Pi OS Trixie Lite
- SocketCAN adapter to vehicle CAN bus
- FTDI USB-RS232 cable to VN300 INS

Display
-------
- 1920 x 720 px LCD display
    - HDMI Interface to Raspberry Pi
